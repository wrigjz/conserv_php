This server can be used to obtain conservation values (similar to ConSurf) for residues in a single protein chain. The method it uses is based on the ConSurf method but is a completely new implimentation. For your target protein provide either a single chain PDB file or give the 4 letter PDB code and the chain ID that you would like a prediction for. (Note: Currently we are taking only single chain PDB structures).
